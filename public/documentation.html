Achievement 2 Project: myFlix

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

№ End-points HTTP Request Method Description
1 /users GET Returns a list of all of the users in the website, by using GET.
2 /users/[:name] GET Allows us to get the data from a single user by their username
3 /updateusers PUT Updates a users username
4 /deleteusers DELETE Deletes a user from the server
5 /movies GET Returns a list of all the available movies
6 movies/[:name] GET Allows us to find a movie by their name
7 /addmovies PUT Allows us to add a movie to a username
8 /removemovies DELETE Allows us to remove a movie from a users account
9 /directors PUT Returns a list of all the available directors in the database
10 /director/[:name] GET Allows us to find a specific director by name.
11 /genres GET Returns a list of all the available genre
12 /genres/[:name] GET Allows us to find a genre by their name
13 /sign-up POST Lets us make our account to access the service
14 /log-in GET Lets us access the account we previously made to use the service
